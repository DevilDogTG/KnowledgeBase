# 📚 Local Knowledge Base

This is the **local knowledge base** — rough working notes, investigations, and
references kept for internal/day-to-day use. Entries here are intentionally low-ceremony:
they don't need polished formatting. When a note is worth publishing, it gets cleaned up
and promoted into the web blog at [`/src`](../src) as a post.

> **Flow:** rough notes live here in `/docs` → a curated subset is polished and published
> to `/src`. Not every note graduates, and `/src` may also contain original posts that
> never lived here.

## Conventions
- One note per file, Markdown.
- Prefer a short `# Title` heading at the top of each note.
- Add new notes to the relevant topic folder and link them from that folder's `Index.md`.
